# Astro Defender

A browser-based implementation of the Astro Defender survival game.

## Features

- Spaceship rotation
- Bullet shooting
- Asteroid spawning
- Asteroid splitting mechanic
- Score tracking
- Collision detection
- Game Over system

## Controls

- Left Arrow Key -> Rotate Left
- Right Arrow Key -> Rotate Right
- Spacebar -> Fire Bullet

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API

## Project Structure

AstroDefender/
├── index.html
├── style.css
├── game.js
└── README.md

## How to Run

1. Open index.html in a browser
OR
2. Use VS Code Live Server
